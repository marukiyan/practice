# Программа обработки изображений

## Описание
Программа выполняет обработку изображений: позволяет загружать изображения с диска, делать снимки с камеры, отображать цветовые каналы, обрезать, рисовать прямоугольники и вращать изображения.

## Репозиторий
Вы можете клонировать этот репозиторий с GitHub:
[https://github.com/marukiyan/practice.git]

## Требования
Программа требует следующие зависимости:
- `opencv-python`
- `numpy`

## Установка зависимостей
Чтобы установить все зависимости, выполните команду:
```bash
pip install -r requirements.txt

## Активируйте виртуальную среду (если используется)
 venv\Scripts\activate

## Запустите программу:
 python app.py
